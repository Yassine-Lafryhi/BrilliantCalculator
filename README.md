* [Swing-GUI]()
## Prerequisites

- Install java :
> On MAC OS using [Homebrew](https://brew.sh/) :
```shell
brew install 
```
> On Linux (Ubuntu) :
```shell
sudo apt-get install 
```
> On Windows using [Chocolatey](https://chocolatey.org/) :
```shell
choco install 
```
